Website Creator

The Website Creator software was created to create a unified website template for this domain. The features are basic, but suffice for my needs, which is a basic HTML page. The programm creates this unified page from your content html page, which should only include basic html code. I strongly recommend dillinger.io for the HTML creation, as it allowes you to write markdown. You can export it then as HTML and also save your drafts on your Google Drive (few more) or download them.

Seriously. I’m out of my data volumes after ten days every month. And then, for every tech article, for every tutorial, for every single ressource beside wikipedia, i need to load 1 MB websites. This takes forever, and it’s mostly junk data like nonnecessary pictures or advertisement. With this website creator, web pages are exactly that. Webpages. There look shitty, but score better than anything else on website performance tests, are tiny in size (19KB for example), and aren’t full of shit. The basic template provides just enough to make the sites readable on mobile devices, and enjoyable on desktops.

Requirements

  • Python3 installed

Usage

For Windows

  • You have to have Python 3 installed
  • Just click it and follow the questions on screen

For Linux/MacOS(X)

  • Start it in terminal ./wscreator <your content html file> <final name>
  • Follow the questions on screen
